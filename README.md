# Практическое занятие 3: Стилизация веб-страницы

## Описание проекта
Стилизация веб-сайта веб-студии с применением методологии БЭМ (Блок-Элемент-Модификатор) и современных подходов CSS.

## Выбранная методология: БЭМ
- БЭМ обеспечивает четкую структуру и самодокументируемость кода
- Позволяет легко масштабировать проект и поддерживать его
- Идеально подходит для командной работы
- Предотвращает конфликты специфичности CSS

## Реализованные функции:
1. **Семантическая HTML-разметка** из Практического занятия 2
2. **Методология БЭМ** для именования CSS-классов
3. **Подключение шрифтов** через Google Fonts (Inter)
4. **CSS-переменные** для цветов, отступов и радиусов
5. **Интерактивные элементы** с состояниями :hover, :focus, :active
6. **Плавная прокрутка** для якорных ссылок
7. **Микро-анимации** и переходы
