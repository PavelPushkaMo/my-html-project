![Структура сайта WebStudio Pro](assets/images/7Dz.png)

##  О проекте
Многостраничный сайт для веб-студии **WebStudio Pro**, разработанный в рамках практического занятия №8 "Создание многостраничного сайта".

##  Структура сайта
- **Главная страница** (`index.html`) - лендинг с услугами, портфолио, информацией о компании
- **Контакты** (`contacts.html`) - форма обратной связи, контактная информация, карта
- **FAQ** (`faq.html`) - часто задаваемые вопросы с аккордеоном и фильтрацией по категориям
- **404** (`404.html`) - креативная страница ошибки с полезными ссылками

##  Использованные технологии
- HTML5
- **CSS-фреймворк: Bulma** (v1.0.4)
- Font Awesome 6 (иконки)
- Google Fonts (Inter)
- Яндекс.Карты (iframe)

##  Адаптивность
- Desktop (широкие экраны)
- Tablet (планшеты)
- Mobile (мобильные телефоны с бургер-меню)
