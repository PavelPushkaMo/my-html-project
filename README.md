# WebStudio Pro - Лендинг веб-студии

##  Карта сайта

![Структура сайта WebStudio Pro](assets/images/7Dz.png)

##  О проекте
Одностраничный лендинг для веб-студии **WebStudio Pro**, разработанный с использованием CSS-фреймворка Bulma.  
Проект создан в рамках практического занятия №7 "Работа с CSS-фреймворками".

##  Тема проекта
Сайт веб-студии, предоставляющей услуги веб-разработки, UI/UX дизайна и SEO-оптимизации.

##  Использованные технологии
- HTML5
- **CSS-фреймворк: Bulma** (v1.0.4)
- Font Awesome 6 (для иконок)
- Google Fonts (Inter)
- Дополнительные кастомные стили

##  Структура лендинга
- ✅ **Хедер** с логотипом и навигацией
- ✅ **Hero-секция** с УТП и CTA-кнопками
- ✅ **Услуги** (3 карточки)
- ✅ **Портфолио** (сетка 3x2 с проектами)
- ✅ **О компании** (миссия, ценности, команда)
- ✅ **Отзывы клиентов**
- ✅ **Форма связи** с валидацией
- ✅ **Футер** с контактами и соцсетями

##  Адаптивность
- Полностью адаптивный дизайн для всех устройств
- Бургер-меню на мобильных
- Оптимизированное отображение карточек
