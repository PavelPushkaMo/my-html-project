# Семантическая верстка сайта

## Описание проекта
Учебный проект по созданию семантической HTML-разметки сайта по макету. Проект выполнен в рамках практического занятия по фронтенд-разработке.

## Используемые технологии
- HTML5
- Семантическая разметка
- Git для контроля версий
- GitHub Pages для хостинга

## Семантические теги, использованные в проекте
- `<header>` - шапка сайта
- `<nav>` - навигационное меню
- `<main>` - основное содержимое
- `<section>` - тематические разделы
- `<article>` - самостоятельные блоки контента
- `<aside>` - боковая панель
- `<footer>` - подвал сайта
- `<h1>`-`<h6>` - иерархия заголовков

## Проверка валидации
HTML-код проверен через [валидатор involta](https://involta.ru/tools/validator-html/) и не содержит критических ошибок.

## Типографика
Текст обработан через онлайн-типограф для корректного отображения кавычек и тире.
